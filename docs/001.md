### 用户信息user_info

| --------- | --- | --- |
| user_id | 用户ID | Long |

| name | 用户姓名 | String |
| age | 年龄 | Integer |
| professional | 职业 | String |
| city | 所在城市 | String |
| sex | 性别 | String |
